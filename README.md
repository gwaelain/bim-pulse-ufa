# BIM Pulse Ufa

Статический сайт для GitHub Pages: BIM, AI, Revit, Dynamo, PostgreSQL, Docker и плагины.

## Файлы
- `index.html` — главная
- `article.html` — страница статьи
- `news.js` — база из 12 статей
- `script.js` — фильтры, карточки, анимации, модальные окна
- `style.css` — дизайн

## Как добавить статью
Открой `news.js` и добавь новый объект в массив `window.NEWS`.

## GitHub Pages
Settings → Pages → Branch `main` → `/root`.
