# BIM Pulse Ufa

Статический сайт для GitHub Pages: BIM, AI, автоматизация, новости и статьи.

## Файлы
- `index.html` — главная страница
- `article.html` — страница статьи
- `news.js` — база статей для страницы статьи
- `script.js` — фильтры, анимации, модальные окна
- `style.css` — адаптивная стилистика

## Важно
Файлы изображений должны лежать в корне репозитория:
- `logo.png`
- `bim-case-1.jpg`
- `bim-case-2.jpg`
- `bim-model-1.png`
- `bim-model-2.png`

## GitHub Pages
Settings → Pages → Branch: `main` → Folder: `/root`.

## Изображения для новостей
К новостям уже привязаны изображения из корня репозитория:
- Bim.png
- Bim1.png
- Bim2.png
- bim-case-1.jpg
- bim-case-2.jpg
- bim-model-1.png
- bim-model-2.png

Чтобы заменить картинку у статьи, открой `news.js` и поменяй поле `image` у нужной новости.
