# BIM Pulse Ufa

Одностраничный сайт для направления BIM в Уфе и Республике Башкортостан.

## Как запустить

1. Распакуйте архив.
2. Откройте `index.html` в браузере.
3. Замените:
   - `https://t.me/your_username` на свой Telegram
   - `yourmail@example.com` на свой email
4. Загрузите файлы в GitHub Pages.

## Файлы

- `index.html` — структура сайта
- `style.css` — дизайн
- `script.js` — плавная прокрутка
