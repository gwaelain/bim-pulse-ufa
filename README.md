## Как обновить GitHub Pages

1. Открой репозиторий: https://github.com/gwaelain/bim-ufa
2. Замени файлы: index.html, style.css, script.js
3. Убедись, что в корне репозитория есть изображения: logo.png, bim-model-1.png, bim-model-2.png
4. Нажми Commit changes.
5. Проверь сайт: https://gwaelain.github.io/bim-ufa/

Если сайт не обновился — нажми Ctrl + F5.
