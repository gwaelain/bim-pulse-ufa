# BIM Pulse Ufa

Статический сайт для GitHub Pages: BIM, AI, Revit, Dynamo, PostgreSQL, Docker и плагины.

## Файлы

- `index.html` — главная страница
- `style.css` — стили и адаптив
- `script.js` — анимации, счётчики и карточки медиа
- `news.js` — данные для блока медиа
- `news.json` — резервный файл с новостями
- `logo.png` — логотип проекта
- `bim-case-1.jpg`, `bim-case-2.jpg` — изображения кейсов
- `bim-model-1.png`, `bim-model-2.png` — дополнительные изображения

## Публикация на GitHub Pages

1. Загрузить файлы в корень репозитория.
2. Открыть `Settings` → `Pages`.
3. Source: `Deploy from a branch`.
4. Branch: `main`, folder: `/root`.
